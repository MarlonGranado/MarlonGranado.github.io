Calculator

This calculator is my first personal project developed with HTML, CSS, and JavaScript to practice event handling and logical development. It can switch between themes (light/dark) and includes various validations to prevent users from entering incorrect operations (so it will never give you an error when clicking the equal button) and provides the result using the resolve() function. Currently, another version is under development with the SOON button that will allow you to perform more complex calculations such as roots, parentheses, powers, percentages, and all types of operations.
